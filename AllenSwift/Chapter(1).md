# Chapter 1. Part 1-6
## Part 1. 변수와 상수

- __변수(var)__
```swift
    var a = 4           // print(a) = 4
    var b = 6           // print(b) = 6
    a = 3               // print(a) = 3
```

즉, 변수는 안의 값이 바뀔 수 있는 것

- __상수(let)__
```swift
    let a = 3
    let b = 5 
    a = b               // 오류 발생
```
즉, 상수는 안의 값이 변할 수 없는 것



- __데이터타입__
```swift
    var a: Int = 4
    a = 3.14            // 오류 발생
```

데이터 타입은 선언할 때 주었던 형태와 맞는 값을 변수로 사용할 수 있다.



- __Character와 String의 차이__
```swift
    let Str: String = ""
    let Cha: Character = " "
```
String은 빈 문자열을 저장할 수 있다.
Charater는 빈 문자열을 저장할 수 없음(공백은 가능)
