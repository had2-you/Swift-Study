# Chapter 1. Part 1-6


### Part 1. 변수와 상수
변수(var)

    var a = 4           // print(a) = 4
    var b = 6           // print(b) = 6
    a = 3               // print(a) = 3
    
즉, 변수는 안의 값이 바뀔 수 있는 것

상수(let)

    let a = 3
    let b = 5 
    a = b               // 오류 발생

즉, 상수는 안의 값이 변할 수 없는 것
